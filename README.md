## Pong JS :video_game:

Se puede jugar 02 personas con en el mismo teclado
| Jugadores   |      Subir      |  Bajar |
|----------|:-------------:|------:|
| Jugador 1 izquierda | tecla (W) | tecla (S) |
| Jugador 2 derecha | pulsar flecha arriba |  pulsar flecha abajo |
